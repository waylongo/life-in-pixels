# Life in Pixels

把人生摊成一张 A4 纸。

960 个格子，每格一个月。它不试图解释人生，只把时间放到眼前：已经走过的、正在发生的、尚未抵达的，都安静地占据自己的位置。

## 它是什么

Life in Pixels 是一个纯静态的人生地图工具。输入出生年月后，页面会生成 80 年、960 个月的 A4 海报，并允许你记录学习、事业、家庭、健康、转折等里程碑。

它适合用来：

- 看见时间的尺度
- 标记重要事件和计划
- 打印或导出一张个人时间地图
- 在有限中重新选择优先级

## 一点想法

我们常把时间想成河流，于是默认它会继续流下去。

但当时间变成格子，它就不再抽象。一个月很小，小到容易被浪费；一生也很小，小到可以被一张纸装下。

这不是倒计时。它只是提醒：生活不是无限展开的背景，而是需要亲手涂色的有限画布。
